## 🎯 What Is This?

**Instagram Engagement Dashboard** is a data analytics project that breaks down performance across **200 Instagram channels** using key engagement metrics — likes, comments, views, followers, and engagement rate. Built in Power BI with Python-powered data preparation, it surfaces the patterns that raw numbers hide: which categories dominate, why mid-size accounts often outperform large ones, and what content strategies actually move the needle.

## 🔍 Key Insights

| # | Finding |
|:--|:--------|
| 📉 | **Follower count ≠ engagement** — as follower counts grow, engagement rate tends to drop |
| 🏆 | **Mid-size channels punch above their weight** — medium-follower accounts consistently outperform mega-accounts |
| 🎬 | **Category is king** — certain content verticals significantly outperform others in average engagement |

## 🚀 Getting Started

### Prerequisites

Before you begin, make sure you have:

- ✅ **Power BI Desktop** — [Download free](https://powerbi.microsoft.com/desktop)
- ✅ **Python 3.8+** *(only needed if re-running data cleaning)*
- ✅ **Pandas** — `pip install pandas`

---

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/instagram-engagement-dashboard.git
cd instagram-engagement-dashboard
```

---

### 2️⃣ Open the Dashboard

- Launch **Power BI Desktop**
- Open `dashboard.pbix`
- Visuals load automatically from the cleaned CSV

---

### 3️⃣ Explore

- Use slicers to filter by **category**, **follower range**, or **engagement tier**
- Hover over charts for detailed tooltips
- Compare channels side by side across all 6 metrics
